# iTeam 功能概览

本目录记录 iTeam 系统的所有功能模块规格说明。每个功能都有唯一的编号，便于需求追溯和版本管理。

## 功能编号规则

- **F001-F099**: 用户与认证模块
- **F100-F199**: 设备管理模块
- **F200-F299**: 项目管理模块
- **F300-F399**: 文档管理模块
- **F400-F499**: 统计与分析模块
- **F500-F599**: 系统设置模块
